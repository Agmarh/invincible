# Effets
Tant que vous portez cette cape avec le capuchon tiré, les créatures qui tentent un test de Perception pour vous voir subissent un désavantage, tandis que vous obtenez l'avantage sur les tests de Discrétion effectués pour vous cacher, car les teintes de la cape se modifient pour vous camoufler au mieux. Il faut une action pour tirer ou rabattre la capuche.
